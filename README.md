# Vatomouro - βατόμουρο(vatómouro) = Greek fpr Raspberry

The purpose of this is more for learning so dont expect anything interessting or new here. Feel Free to offer advise or critiscm.

- 1 Raspberry Pi 4 Mod. B

- old Ethernet-switch

Software:
